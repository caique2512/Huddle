# Huddle

Projeto frontend simples desenvolvido como um exercício de layout responsivo. A ideia é recriar a tela inicial de um produto hipotético chamado **Huddle**, inspirado em designs de landing page com ilustrações e chamadas para ação.

## Estrutura do projeto

```
index.html            # Página principal
src/
  reset.css           # Reset de estilos básicos
  estilos.css         # Estilos personalizados
images/               # Imagens utilizadas na página
```

> O repositório foi criado no Windows e contém apenas HTML e CSS.

## Como usar

1. Abra o arquivo `index.html` em um navegador moderno.
2. Edite `src/estilos.css` para ajustar cores, larguras ou comportamento responsivo.

O layout principal utiliza flexbox para posicionar a ilustração e o bloco de conteúdo lado a lado.

## Conteúdo

- Cabeçalho com logo
- Seção principal com imagem ilustrativa e texto (título, subtítulo e botão)
- Rodapé com ícones de redes sociais

## Observações

Este projeto é uma base estática; não possui back-end nem dependências. É ideal para estudo de CSS e posicionamento.
